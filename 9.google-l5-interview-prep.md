## 💻 DSA Problem Framework (UMPIRE)

```
U — Understand    → Problem বোঝো, clarify করো, examples নাও
M — Match         → কোন data structure/pattern match করে?
P — Plan          → Pseudo-code লেখো আগে
I — Implement     → Code লেখো
R — Review        → Edge cases check করো
E — Evaluate      → Time/Space complexity বলো
```

### Common Patterns to Master

```
Array/String:    Two pointers, Sliding window, Prefix sum
Tree/Graph:      DFS, BFS, Topological sort
Dynamic Prog:    Memoization → Tabulation
Heap:            K-th element problems
HashMap:         Frequency count, Two-sum pattern
Binary Search:   Search in rotated array, range problems
```

### Time/Space Complexity — Instant Recall

```
O(1)      → HashMap lookup, array index
O(log n)  → Binary search, balanced BST
O(n)      → Linear scan, HashMap build
O(n log n)→ Sorting (merge sort, heap sort)
O(n²)     → Nested loops, bubble sort
O(2ⁿ)     → Recursive subsets, combinations
```

## 🌐 Frontend-Specific Interview Topics

### Browser & DOM
```
- Event loop, microtask vs macrotask queue
- Critical rendering path
- Reflow vs repaint — কখন trigger হয়?
- Virtual DOM vs real DOM trade-offs
- Web Workers, Service Workers পার্থক্য
```

### Performance
```
- Core Web Vitals: LCP, INP, CLS — কী এবং কীভাবে improve করে
- Bundle size optimization: tree shaking, code splitting, lazy loading
- Image optimization: WebP, lazy loading, srcset
- Caching strategies: HTTP cache, Service Worker cache, Ember Data backgroundReload
```

### JavaScript Deep Dive
```
- Closures, scope chain, hoisting
- Prototype chain, inheritance patterns
- Event delegation vs direct binding
- Memory leaks — কীভাবে হয়, কীভাবে ধরো
- async/await internals — microtask queue
- WeakMap/WeakRef — memory-sensitive data structures
```

### Ember.js / Framework Questions
```
- Glimmer rendering engine কীভাবে কাজ করে?
- @tracked reactivity — কীভাবে change detection হয়? (dirty checking নয়, Proxy-based)
- @cached — কীভাবে memoization কাজ করে?
- TrackedArray vs plain @tracked array — পার্থক্য কী?
- Ember Data store — identity map, cache strategy, backgroundReload
- Route lifecycle hooks — কোনটা কখন call হয়? (beforeModel → model → afterModel → setupController)
- Loading/error substates — কীভাবে auto-activate হয়?
- ember-concurrency — কেন raw async methods avoid করবে? (lifecycle, cancellation)
- GJS export rules — কেন default export mandatory?
```
